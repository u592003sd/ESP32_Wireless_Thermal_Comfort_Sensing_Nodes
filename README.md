# ESP32 Based Wireless Thermal Confort Sensing Nodes
- Created a ESP32 shield compatible with DS18B20 Temp. Sensor & DHT11 RH Sensor. (Best bang for buck)
- Created a ESP32 shield compatible with HDC1008 Temperature Sensor which has 0.2°C Accuracy.
- Optimized routing to 1 layer PCB to match college fabrication facility guidelines.
- Project Documentation folder can be used to refer to System Design Choices and Thermal Sensing Experimentation
